# SangSangPlayground

상상놀이터(유아 교육 앱) 랜딩 페이지 프로젝트입니다.

## 프로젝트 소개

- 유아 대상 창의력/표현력 중심의 교육 앱 소개 페이지
- 히어로 로딩 애니메이션 및 섹션 스크롤 애니메이션 적용
- 요금제, 후기, FAQ, 다운로드 CTA 포함

## 기술 스택

- HTML
- Tailwind CSS (CDN)
- Vanilla JavaScript

## 배포 URL

- https://sunburst77.github.io/SangSangPlayground/

## 실행 방법

정적 페이지이므로 `index.html`을 브라우저에서 열면 바로 확인할 수 있습니다.
