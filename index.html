CORE FEATURES

아이의 잠재력을 깨우는
6가지 핵심 기능

전문 유아교육 연구진이 설계한 놀이 기반 학습 프로그램으로 아이의 창의력, 사고력, 표현력을 골고루 키워줍니다.

자유 그리기 캔버스

100가지 이상의 브러시와 색상으로 아이만의 그림을 자유롭게 그려볼 수 있어요. AI가 그림을 분석해 칭찬 피드백을 줍니다.

리듬 놀이 교실

박자와 멜로디를 직접 만들며 음악적 감각을 키워요. 다양한 악기 소리와 함께하는 음악 창작 놀이.

이야기 만들기

캐릭터를 선택하고 장면을 구성해 나만의 동화책을 완성해요. 상상력과 언어 표현력이 함께 자라납니다.

블록 조립 놀이

3D 블록을 쌓고 조립하며 공간 감각과 논리적 사고력을 키워요. 단계별 미션으로 성취감도 함께!

퀴즈 탐험대

과학, 자연, 생활 속 퀴즈를 풀며 호기심을 자극해요. 정답을 맞힐 때마다 보상 스티커를 모을 수 있어요.

AI 학습 친구

아이의 학습 패턴을 분석해 맞춤형 콘텐츠를 추천해요. 똑똑한 AI 친구가 아이의 성장을 함께합니다.

FAQ

자주 묻는 질문

어떤 연령대의 아이에게 적합한가요?

상상놀이터는 3세부터 7세까지의 유아를 대상으로 설계되었습니다. 연령별로 씨앗반(3-4세), 새싹반(5-6세), 나무반(6-7세) 맞춤 커리큘럼이 제공되며, 아이의 발달 수준에 따라 난이도가 자동 조절됩니다.

하루에 얼마나 사용하는 게 좋을까요?

유아 전문가들은 하루 15~30분 정도의 사용을 권장합니다. 앱 내 사용 시간 제한 기능을 활용하시면 부모님이 직접 일일 사용 시간을 설정할 수 있어요. 시간이 되면 아이에게 친절한 안내 메시지가 표시됩니다.

인터넷 없이도 사용할 수 있나요?

연간 구독 플랜에서는 오프라인 학습 모드를 지원합니다. 미리 콘텐츠를 다운로드해두면 인터넷 연결 없이도 그리기, 블록 조립, 퀴즈 등 주요 기능을 이용할 수 있어요.

구독을 취소하면 데이터가 사라지나요?

아닙니다! 구독을 취소하더라도 아이의 학습 기록, 작품, 성장 리포트는 최대 1년간 안전하게 보관됩니다. 언제든 다시 구독하시면 이전 데이터를 그대로 이어서 사용할 수 있어요.

아이의 개인정보는 안전한가요?

상상놀이터는 COPPA(아동 온라인 개인정보보호법)를 준수하며, 모든 데이터는 국내 서버에 암호화하여 저장합니다. 제3자에게 데이터를 공유하지 않으며, 광고 추적 기술을 일체 사용하지 않습니다.
